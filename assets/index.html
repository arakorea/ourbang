ourbang은 현실 공간을 스캔해 웹에서 탐색 가능한 디지털 공간으로 전환합니다. 장기적으로는 공유·기록·거래까지 확장 가능한 “디지털 공간 단위”를 만듭니다. 시작은 LiDAR 또는 카메라 기반입니다.

지금 탐색하기 (Explore) 스캔 방법 보기 (Scan) 미리보기
현재 버전은 “가능성”을 보여주는 MVP입니다. 자동 3D 생성·인증·결제 등은 기술 성숙도와 파트너십에 따라 단계적으로 통합됩니다.

FAQ

LiDAR가 꼭 필요하나요?

아닙니다. MVP는 카메라 기반 플로우도 전제로 합니다. 다각도 촬영 → (서비스 측) 모델 생성의 형태로도 운영 가능합니다.

FAQ

어떤 결과물이 나오나요?

웹에서 탐색 가능한 디지털 공간(뷰)과, 향후 관리·공유·거래를 위한 ‘공간 단위’ 메타데이터를 목표로 합니다.

FAQ

다음 단계는 무엇인가요?

스캔 → 생성 → 검수 → 공개의 파이프라인 고도화와, 외부 파트너(촬영/생성/유통/결제) 연동이 핵심입니다.
