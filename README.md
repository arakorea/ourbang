# ourbang

**Scan your space. Experience it virtually. Own it digitally.**

ourbang은 현실 공간을 스캔해 가상으로 경험하고, 디지털 공간으로 관리하는 공간 기반 플랫폼입니다.  
LiDAR 또는 카메라 기반으로 시작하며, 기술 발전과 파트너십에 따라 자동화/확장 기능을 단계적으로 통합합니다.

---

## What is ourbang?

ourbang은 단순한 3D 뷰어가 아니라, 공간을 ‘경험하고 관리할 수 있는 단위’로 재정의합니다.

- 개인 공간: 방, 작업실, 스튜디오
- 상업 공간: 상점, 팝업, 쇼룸
- 기록 공간: 전시, 행사, 임시 공간

---

## Core Flow

1. **Scan** — LiDAR 또는 카메라 촬영으로 공간을 수집  
2. **Explore** — 설치 없이 웹에서 탐색 가능한 경험 제공  
3. **Digital Space** — 보관/공유/거래까지 확장 가능한 디지털 단위

---

## Pages

- `index.html` — 서비스 소개 및 컨셉
- `explore.html` — 가상 공간 탐색 경험
- `scan.html` — 공간 생성(촬영) 가이드
- `vision.html` — 비전/확장 시나리오
